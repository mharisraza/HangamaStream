# HangamaStream - Server
The main backend/API for HangamaStream.

### Stack
- Java 17
- Spring Boot 3.0
- Spring Data
- Spring Security
- MySQL

### What does it do?
This API is responsible for retrieving movies/series from the HangamaStream's MySQL database. It's also responsible for user authentication..
